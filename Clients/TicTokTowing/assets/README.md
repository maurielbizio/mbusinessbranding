# Assets — Tick Tock Towing

Store all brand and visual files here.

## What belongs here
- Logo files (SVG, PNG, AI — vector preferred)
- Brand color swatches or style guide
- Truck photos
- Team photos
- Job site photos
- Any existing marketing materials

## Naming convention
- `logo-primary.svg`
- `logo-white.svg`
- `logo-dark.svg`
- `truck-01.jpg`, `truck-02.jpg`, etc.
- `team-photo.jpg`
